Dynamic 3-Statement Financial Model & Executive SaaS Dashboard (Power BI)
Project Overview

This project is a fully integrated 3-statement financial model built in Power BI using:

Power Query (ETL)
Star schema data modeling
DAX measures
Dynamic financial statements
SaaS KPI analysis
Executive dashboard reporting

The model simulates the financial performance of a SaaS business between 2023–2025 and connects:

Income Statement
Balance Sheet
Cash Flow Statement

into one dynamic reporting environment.

Key Insights Generated
Burn rate increased by approx. 8.4%
EBITDA Margin declined by approx. 7.9%
Employee costs increased significantly over the 3-year period
ARR increased while fixed costs remained stable, indicating scalability potential
